Wie funktioniert der Rundgang?

Schilder an den Stationen

An jeder Station finden Sie ein solches Schild. In der Mitte sehen Sie einen sogenannten QR-Code, den Sie mit Ihrem Smartphone einscannen können.

QR-Code scannen
Öffnen Sie Ihre Kamera-App und Scannen Sie den Code ein. Folgen Sie dem Link, der auf Ihrem Display angezeigt wird - das war's schon!

Es erscheint kein Link?
Dann unterstützt die Kamera-App Ihres Smartphones diese Funktion vermutlich noch nicht. In diesem Fall können Sie sich einen QR-Code-Reader als App kostenfrei herunerladen. Besuchen Sie dazu Ihren App-Store und geben Sie "QR Reader" in die Suche ein.

Kein Datenvolumen mehr?
Rund um den Marktplatz können Sie das freie WLAN der Stadt Amöneburg nutzen. Mehr Infos dazu gibt es hier: Free Wifi der Stadt Amöneburg
